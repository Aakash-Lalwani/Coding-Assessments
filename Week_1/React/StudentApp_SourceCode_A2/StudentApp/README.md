# StudentApp

## Description
StudentApp is a small React assignment that shows a simple student management portal layout with separate Home, About, and Contact sections.

## Features
- Simple landing page layout
- Separate Home, About, and Contact sections
- Basic responsive styling

## Tech Stack
- React
- JavaScript
- CSS
- Create React App

## Installation
1. Open the `StudentApp` folder in a terminal.
2. Run `npm install`.

## How to Run
- Start the development server: `npm start`
- Build for production: `npm run build`

## Assumptions or Limitations
- The content is static and does not use routing or a backend.
- The app is intentionally simple to match a student assignment.
