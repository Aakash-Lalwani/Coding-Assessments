Task 2: Custom Validators & FormArray

Dynamic FormArray with add/remove functionality and typed getter

Custom Validators

  • noCourseCode (sync): Returns { noCourseCode: true } if value starts with 'XX'
  • simulateEmailCheck (async): Returns Promise that resolves after 800ms with { emailTaken: true } if email contains 'test@'

FormArray for Dynamic Controls

  • "Add Another Course" button pushes new FormControl into the array
  • "Remove" button removes a control at specified index
  • Typed getter get additionalCourses(): FormArray provides type safety

Expected Outcome

Entering 'XX101' in courseId shows custom error. test@example.com shows 'Email taken' after 800ms. Additional course controls can be added and removed dynamically. Submit button enables only when all validators pass.
