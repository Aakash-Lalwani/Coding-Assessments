Task 1: Structural Directives

*ngIf with else, *ngFor with trackBy, *ngSwitch for grade status badges

Key Implementations

  • *ngIf: Shows loading message for 1.5s, then displays course grid
  • *ngIf with else: Displays "No courses available" template when array is empty
  • *ngFor with trackBy: Uses trackByCourseId() to only re-render changed items
  • *ngSwitch: Displays different colored badges for passed/failed/pending status

Expected Outcome

Loading message shows for 1.5 seconds then disappears. trackBy is implemented. Switch renders correct badge per grade status. Empty array shows the no-courses template.

Task 2: Attribute Directives (ngClass, ngStyle)

ngClass with getter, ngStyle for dynamic border colors
  • [ngClass]: Applies 'card--enrolled' and 'card--full' classes conditionally
  • [ngStyle]: Sets left border color dynamically based on grade status (green/red/grey)
  • Getter pattern: cardClasses getter keeps template clean by centralizing logic in component class
